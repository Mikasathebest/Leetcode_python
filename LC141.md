141. Linked List Cycle
Easy

题目：判断一个链表是否有环

思路 1：用快慢两个指针，如果有环，两指针会相遇；否则，快指针会指到None
代码如下：

```python
def hasCycle(self,head):
    fast = slow = head
    while fast and slow and slow.next:
        fast = fast.next.next
        slow = slow.next
        if fast == slow:
            return True
    return False
#O(n)
```
思路2：用一个set记录见过的节点，如果节点有重复，则有环；否则，无环
代码如下：
```python
def hasCycle(self,head):
    seen = set()
    while head and head.next:
        if head in seen:
            return True
        else:
            seen.add(head)    
        head = head.next
    return False
    #O(n)    
```
